Capability 02

Auto-Evol Sector Skills

We anchor evaluation on GDPVal — real, paid professional work across dozens of occupations — then build a grader, grow self-evolving Sector Skills, and co-create with practitioners to win in real job scenarios.

Part 01

From GAIA to GDPVal

GDPVal benchmark

It's a level-up, not a side-step. GAIA scores general agent ability — web search, multi-step reasoning, tool use — and we ranked #1. But a top GAIA score still can't tell you whether an agent can finish a real analyst's or marketer's deliverable. So we re-anchored evaluation on GDPVal, OpenAI's benchmark of genuine, paid professional work.

Each task is modeled on a real deliverable and graded against expert human output — economically meaningful work, not toy puzzles. Winning here means winning the job.

Part 02

An automated grader

Try our Grader

GDPVal ships the tasks and grading guidance, but no ready-to-run grader. We built the missing evaluation layer — two complementary graders on one shared harness.

Pairwise GraderBlind A/B comparison of two deliverables with written justification — captures which output a human would actually prefer.
Rubric ScorerCriterion-by-criterion scoring against GDPVal rubrics — gives an absolute, explainable quality number.

Universal Evaluation HarnessOne agent-agnostic pipeline drives both graders, so any agent, product or version is measured the same way — and stays comparable over time.
